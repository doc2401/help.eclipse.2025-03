Launch Bar

Core Build System (CBS) projects rely on the Launch Bar. Make sure the LaunchBar UI feature is installed and enabled. In the global preferences the launch bar can be enabled or disabled.

The launch bar exists out of several components:

  1. Build button, to launch a build.

  2. Run button, to launch a run. This button will change to a Debug button in Debug mode.

  3. Stop button, to stop a Run or Debug session.

  4. Launch mode selector. To change between Run and Debug mode.

  5. Launch configuration selector.

  6. Edit launch configuration. To change the launch configuration properties.

  7. Target selector. Not always visible.

  8. Edit target.
