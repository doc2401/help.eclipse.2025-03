%ExtPoint.quickAssistProcessors

Identifier:
org.eclipse.ui.genericeditor.quickAssistProcessors

Since:
1.1.600

Description:
This extension point is used to contribute quick assist processors for quick content assist support to a given content type.

Configuration Markup:

<!ELEMENT extension (quickAssistProcessor)+>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT quickAssistProcessor EMPTY>

<!ATTLIST quickAssistProcessor

class       CDATA #REQUIRED

contentType IDREF #REQUIRED>

  • class -
  • contentType - The target content-type for this extension. Content-types are defined as extension to the org.eclipse.core.contenttype.contentTypes extension point.

Examples:

API Information:
This extension point does not require specific API's to use. Instead, it reuses IQuickAssistProcessor from org.jface.text.quickassist
